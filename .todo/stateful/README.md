# Components

Litdoc allows you to create interactive components.

- Out of the box data storage and message queue.
- Use plain functions as actions.
- Markdown syntax for adding action buttons.
- Dynamic documentation for actions.
- Markdown and Preact support for building the UI.
- Dispatch actions from Preact components.
- Action bar for quick access to actions.
- Time travel with human readable history.
- Persist state to local storage.
