@lambda-ui/monaco

Monaco Editor is the code editor that powers VS Code. It is a standalone code editor that can be used in any JavaScript application. This package provides Preact hooks for using Monaco Editor.

Hook: useMonacoEditorTheme

The useMonacoEditorTheme hook allows to create a theme for the editor.

Note: The theme is applied to all editors. This is an architectural limitation of Monaco Editor.

Hook: useMonacoEditorSize

The useMonacoEditorSize hook returns the size of the editor content.

Hook: useMonacoEditorValue

The useMonacoEditorValue hook allows to manipulate the editor value.
